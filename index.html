What are Wireframes?

Wireframes are the basic outline of a webpage/application. The reason we use wireframes is so that we have something to follow while we make an app so that we don't get off track or if we work in teams so everyones webpage looks the same

Git Workflow

Git Workflow allow every person in a group to work on a repo. The way it does so is that it allows us to make sepparate branches so that we can all code from different computers. From there we can pull from the master so that everyone has the same things. Later we can merge branches to the master so that we can progress.
